# mouse-event

## Learning Objectives
- Understand how to replicate a mouse event application which is a simple app that shows the the mouse position on the viewport of your device displaying the X and Y axis of the mouse in pixel. it  supports user interface and interesting user experience.

## Description of the Project
For the first milestone in the process of creating mouse event, you will:
- set up a new repository and prepare it for development using best practices.
